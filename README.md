# UK-healthcare-analysis
Healthcare analytics project using SQL, Python, Pandas, Matplotlib, and Seaborn to analyze NHS provider and waiting list dataset

A data analytics project exploring NHS healthcare provider and waiting list datasets using **SQL**, **Python**, **Pandas**, **Matplotlib**, and **Seaborn**.

The project demonstrates how SQL and Python can be combined to clean, analyze, and visualize large healthcare datasets to generate meaningful insights.

---

##  Project Objectives

- Analyze NHS healthcare provider data
- Explore provider distribution across regions and cities
- Identify hospitals with large waiting lists
- Compare waiting lists across treatment departments
- Visualize healthcare data using Python

---

##  Dataset

The analysis uses publicly available NHS datasets including:

- NHS Provider Dataset
- NHS Referral to Treatment (RTT) Waiting List Dataset

---

##  Tech Stack

- Python
- SQL (MySQL)
- Pandas
- Matplotlib
- Seaborn
- VS Code
- DBeaver

---

##  Project Workflow

1. Imported NHS datasets
2. Cleaned missing and unnecessary columns
3. Performed SQL aggregation and filtering
4. Conducted exploratory data analysis (EDA)
5. Built visualizations to identify trends
6. Generated insights from waiting list and provider data

---

##  Visualizations

The project includes:

- Top NHS Providers
- Regional Provider Distribution
- City-wise Provider Distribution
- NHS Waiting List Analysis
- Department-wise Waiting List Analysis
- NHS Heatmap (Hospital vs Treatment Function)

---

##  Key Insights

- London contains the highest concentration of NHS providers.
- Waiting lists vary significantly across treatment functions.
- Large NHS Foundation Trusts contribute a major share of patient waiting lists.
- Heatmap analysis highlights departments with the greatest patient demand.
